# LixDDS

#### 介绍
用Rust开发一个可以应用于跨平台环境的DDS系统。  
欢迎对DDS、Rust感兴趣的人员参与。

#### 标准
1. 《Data Distribution Service (DDS)》Version 1.4  
2. 《The Real-time Publish-Subscribe Protocol DDS Interoperability Wire Protocol (DDSI-RTPS) Specification》Version 2.3
